Overview

arac is a library for neural networking applications with the following in mind:
  • As fast as possible without ugly code.
  • As flexible as performance allows.
  • Glue it together - use high level languages to solve your problem without losing the speed.

Features

Currently, the following is included:
  • Feedforward, recurrent and multi-dimensional recurrent neural networks with long short-term memory (LSTM) cells.
  • Backprop and backprop through time training
  • Datasets of various forms
Furthermore, all algorithms are usable either via C++ or Python via PyBrain.

Dependencies

arac uses scons for building and LAPACK for fast linear algebra operations.

Download

You can download this project in either zip or tar formats. You can also clone the project with Git by running: 
$ git clone git://github.com/bayerj/arac

Documentation

Doxygen documentation files are included with the distribution and can be built by running the following command the following command: 
 $ arac-src/docs/ doxygen Doxyfile
There are no tutorials at this point.
